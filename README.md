# 🌙 مسبحة - السبحة الإسلامية الذكية
## Masbaha - Smart Islamic Tasbeeh Counter

تطبيق ويب متكامل وحديث لعداد التسبيح والأذكار الإسلامية مع ميزات متقدمة وتجربة مستخدم سلسة.

A comprehensive and modern web application for Islamic Tasbeeh counter and Dhikr (remembrance) with advanced features and smooth user experience.

---

## ✨ الميزات الرئيسية | Main Features

### 🔢 العداد الذكي | Smart Counter
- ✅ عداد تسبيح سلس وسريع الاستجابة
- ✅ تاريخ العمليات (التراجع)
- ✅ أزرار سريعة للإضافة (+10, +33, +99, +100)
- ✅ دعم متعدد الأذكار
- ✅ اهتزاز وصوت عند الضغط

### 📈 تحديد الأهداف | Goal Setting
- 🎯 تعيين هدف يومي للتسبيح
- 📊 شريط تقدم بصري
- 🔔 إشعارات عند الوصول للهدف
- 📅 إحصائيات يومية وعامة

### 📖 مكتبة الأذكار | Dhikr Library
- سبحان الله
- الحمد لله
- الله أكبر
- لا إله إلا الله
- أستغفر الله
- أذكار الصباح والمساء مع المراجع الشرعية

### ⚙️ الإعدادات المتقدمة | Advanced Settings
- 🎨 مظهر فاتح/داكن مع وضع تلقائي
- 🔊 تحكم في الصوت والاهتزاز
- 🔤 تحديد حجم الخط (صغير، متوسط، كبير، كبير جداً)
- 📤 تصدير واستيراد البيانات
- 📱 واجهة مسؤولة (Responsive)

### 💾 حفظ البيانات | Data Management
- 💿 حفظ تلقائي في LocalStorage
- 📊 إحصائيات تفصيلية
- 🔄 تصدير البيانات بصيغة JSON
- 📥 استيراد البيانات من ملفات سابقة

### 📱 تطبيق ويب تقدمي | Progressive Web App (PWA)
- 📲 يمكن تثبيته كتطبيق على الهاتف
- 🔌 يعمل بدون اتصال بالإنترنت
- ⚡ سرعة تحميل فائقة
- 🎯 اختصارات سريعة

## 🛠️ التكنولوجيا المستخدمة | Technologies Used

- **HTML5** - الهيكل
- **CSS3** - التصميم مع دعم Dark Mode
- **Vanilla JavaScript** - البرمجة الوظيفية
- **Service Workers** - للعمل بدون إنترنت
- **Web API** - Notifications, Vibration, Audio
- **LocalStorage** - حفظ البيانات
- **PWA** - تثبيت التطبيق

## 🚀 البدء السريع | Quick Start

### الطريقة 1: استخدام مباشر
1. انسخ ملفات المشروع
2. افتح `index.html` في المتصفح
3. ابدأ التسبيح! 🤲

### الطريقة 2: تثبيت كتطبيق PWA
1. افتح الموقع في المتصفح
2. اضغط على قائمة المتصفح (الثلاث نقاط)
3. اختر "تثبيت التطبيق" أو "Add to Home Screen"

### الطريقة 3: استضافة محلية
```bash
# استخدام Python
python -m http.server 8000

# أو استخدام Node.js
npm install -g http-server
http-server
```

ثم افتح `http://localhost:8000`

## 💡 كيفية الاستخدام | How to Use

### العداد | Counter
1. اختر الذكر من القائمة المنسدلة
2. اضغط الزر "+" أو اضغط مفتاح المسافة
3. استخدم الأزرار السريعة للإضافات المتعددة
4. اضغط "تراجع" للرجوع للعدد السابق

### تحديد الهدف | Set Goal
1. أدخل العدد المطلوب في حقل الهدف
2. اضغط "تعيين الهدف"
3. سيظهر شريط التقدم
4. ستتلقى إشعار عند الوصول للهدف

### الإعدادات | Settings
- غيّر المظهر (فاتح/داكن)
- فعّل/عطّل الصوت والاهتزاز
- كبّر حجم الخط حسب الحاجة
- صدّر بياناتك أو استورد ملفات قديمة
